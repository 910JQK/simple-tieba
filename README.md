# 极简贴吧

> 本程序为 Android 开发实验程序，不作任何担保，使用者因使用本程序造成的损失开发者概不负责。使用本程序发出的回复容易被*度删除，假如帖子频繁被删，请使用右上角菜单中的「用浏览器打开」功能，在浏览器中进行回帖。

## Build

```console
$ cd simple-tieba
$ yarn build
$ cd ..
$ cordova platform add android
$ cordova run android
```
